## Features Overview

This release provides compatibility with the following versions of the Open Banking Read/Write Data API
Specification *(3.1.11, 3.1.10, 3.1.9, 3.1.8, 3.1.7, 3.1.6, 3.1.5, 3.1.4, 3.1.3, 3.1.2, 3.1.1 and 3.1.0)*.
